Hello from Chimera!

This site is hosted on the Chimera P2P network.

No servers. No cloud. Just peers.

How it works:

  • Files are split into 256KB chunks
  • Each chunk is SHA-256 hashed for integrity
  • A Merkle DAG maps the file structure
  • Peers discover content via Kademlia DHT
  • Browsing a site = caching and seeding it
